# StudyVault

A basic starter Obsidian vault for students.

Most vault templates are convoluted and difficult to manage. StudyVault keeps things simple by using a modified PARA method, breaking your vault down into:
- **Areas** — summaries and key notes on important topics
- **Projects** — time-sensitive, short-term pieces of work such as rotations, modules or attachments
- **Resources** — atomic, Zettelkasten-style supporting notes that provide scaffolding that connects your vault
- **System** — the backend of the vault, including templates, master lists, and supporting files
- **Tasks** — a living tracker for active to-dos

## Includes
- Core structure
- Demo note
- Dataview query for homepage motivation

## Excludes
- Example project and resource notes — these will be added in the future

## Instructions
This will be explained in more detail in the future. This is just a quick scaffold for people to test for study purposes. 

You will need to:
- Set up **Obsidian Git** for backup and version control
- Install **LearnKit** and add an API key if you want AI features  
  - This can be done for free using OpenRouter

## Demo flashcards
Some demo LearnKit flashcards are included on **Demo One** in **Areas**.

You will need to sync these with LearnKit before you can study them. Do this via the **Command Palette** in Obsidian using the command "Sync all flashcards from the vault".
